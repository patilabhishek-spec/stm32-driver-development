# STM32 Driver Development

## Description
This project contains low-level driver development for STM32F446xx microcontroller using Embedded C.

## Drivers Implemented
- GPIO Driver
- SPI Driver
- I2C Driver
- USART Driver
- RCC Driver

## Features
- Register-level programming (no HAL used)
- Modular driver design
- Easy to integrate with applications

## Folder Structure
- Header files (.h)
- Source files (.c)

## Application
- Embedded systems development
- Learning microcontroller peripherals
